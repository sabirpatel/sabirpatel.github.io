Technologies Used

  1. Dreamweaver was used to develop the web pages.
  2. Digital Cameras were used for video and still pictures.
  3. Gimp and SnagIt were used to manipulate images.
  4. Any Video Converter was used to compress videos.
  5. Windows MovieMaker was used to edit videos.
  6. jQuery was used to create accordions.
  7. The prettyPhoto jQuery plugin was used as a video player and image viewer.
  8. Quick Screen Capture and SnagIt were used for screen captures.
